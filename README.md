# Real-Time Classroom Assistance System

This project is a real-time classroom platform where students can ask questions and teachers can respond instantly.

## Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB  
Real-time Communication: Socket.io

## Features
- User Registration and Login
- Create Classroom
- Join Classroom
- Ask Questions in Real Time
- Teacher Response System

## How to Run

Backend:
cd backend
npm install
npm start

Frontend:
cd frontend
npm install
npm start
